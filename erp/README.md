# 术语定义
| 名称 | 定义 | 说明 |
| :----: | ---- | ---- |
| ERP | 企业资源计划 |  |
| SCM | 供应链管理 | 进销存管理。着重各个流程的监控，属于过程监管软件，过程中发生问题可以及时解决，挽回损失 |
| PSS, Purchase Sale Stock | 进销存 | 是SCM的一部分。结果型软件，基本上用于记录的，只能事后发现问题后补救 |
| WMS | 仓储管理 |  |
| CRM | 客户关系管理 |  |
